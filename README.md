# SUDO DND

## Como executar

Para executar a aplicação existem 2 modos:

### Docker (Preferível)

A execução pelo Docker consiste em 2 passos.

Primeiro, construa a imagem da aplicação com o comando:
```sh
docker build -t sudo-dnd .
```

Após isso, basta executar o container com o comando:
```sh
docker run -it sudo-dnd
```
